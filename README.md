## ✨ Overview

**OpenFlow** is a clean and powerful web app that lets you interact with multiple AI models using enhanced system prompts designed to unlock more flexible and unrestricted responses.

No complicated setup.  
No paywalls to start.  

Just Run, add your API key, and go.

---

## 🚀 Features

- 🔐 **Authentication System**
  - Secure login & account management

- 🤖 **Multi-AI Access**
  - Claude  
  - ChatGPT (in development)
  - Gemini  
  - Grok  

- 🧠 **Built-in Jailbreak Prompts**
  - Automatically applied for enhanced outputs

- 🔑 **OpenRouter API Integration**
  - Bring your own key (free tier supported)

- ⚡ **Fast & Minimal UI**
  - Clean, distraction-free experience

---

## 🛠️ How It Works

```text
1. Download source and run
2. Add your OpenRouter API key
3. Select an AI model
4. Start chatting (jailbreak prompts applied automatically)
```

---

## 🔑 Get Your API Key (Free)

1. Visit: https://openrouter.ai  
2. Download source and run
3. Generate an API key  
4. Paste it into OpenFlow website when ran

---

## 📦 Installation

```bash
# Clone the repository
git clone https://github.com/bigcheesh/openflow

# Navigate into the folder
cd openflow

# Install dependencies
npm install

# Run the development server
npm run dev
```

## ⚙️ Environment Setup

`.env` file in the root directory:

```env
OPENROUTEAPI=OPENROUTE API
```

> Add additional environment variables depending on your backend or auth setup.

OpenFlow includes **jailbreak-style prompts** intended to explore AI capabilities beyond default constraints.

- Use responsibly

- AI providers may change behavior or restrict usage at any time
